# Linkee - minimal URL shortener
Linkee is a dead simple URL shortner built with NextJS and Appwrite. You can generate short links and get analytics about them without signing up.

## Features
- Generate short links
- View click analytics (total clicks, referrers, browsers, OS, countries)
- No sign-up required
- Built with NextJS and Appwrite
- Open source

You can try out a live demo of Linkee [here](https://linkee.ultimatedev.space).
